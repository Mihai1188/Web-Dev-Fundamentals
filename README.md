# Web-Dev-Fundamentals

Web Development Fundamentals

Project

This repository contains my solutions for the Web Development I assignment.
The project demonstrates fundamental web development techniques using HTML5 and CSS.

Exercises

Exercise 1 – Semantic Markup

This exercise shows how to use semantic HTML5 elements to structure your webpage.
Tags like as header, nav, main, section, article, and footer have been used to organise content and provide better accessibility.

Exercise 2 – CSS Styling

This exercise adds CSS styling to an HTML page.
An external stylesheet was used for colours, typography, spacing, and layout improvements.

Exercise 3 – Page Layout

This exercise shows page layout posibilities using Flexbox.
The layout has a navigation bar, a main content area, and a sidebar.

Exercise 4 – Web Accessibility

This exercise demonstrates accessibility features including:
	•	Text for images
	•	Form labels
	•	Semantic HTML structure
	•	Navigation

Technologies Used
	•	HTML5
	•	CSS3
	•	Flexbox
